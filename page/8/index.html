Ruby Tuesday 五月場

這周參加了五月場的 Ruby Tuesday，
之前在台中都沒機會參加，
好不容易現在搬回台北，一定要去一下的。

這次兩個講題分別是 kevin 分享他們公司使用
flex + RoR 開發影音串流網站的心得。

由於和之前在維格開發的網站概念很類似，
因此對這個主題特別有興趣，也提了一些問題。

他們目前 beta 階段，scalibility 預估承載量是 100 channels * 50000 users.
也就是同時五百萬人上線觀看，
farm的規模目前是16台機器，
聽到這邊，回想起那時開發到需要一台 server run 影音轉碼+測 streaming
老闆連一台 server 都不肯買…最後做不出什麼東西也是剛好而已…。
順帶一提，kevin 他們的人員編制是 6 個 RD + 6 個設計，開發了三個月，
人力是我們的當時的 3 倍。

ihower 介紹了自己開發的一些 helper ，感覺都很實用，
也提到了 http://ennerchi.com/projects/jrails 這個 RoR 的 jquery plugin，
之後有空都要試試。

在 ihower 的 blog 可以抓到這次的 slides。

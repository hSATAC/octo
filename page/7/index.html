臨界點

Aug 31st, 2008

快爆炸了。

搬回台北後，沒一件好事。
首先是莫名其妙的咳嗽不停，
咳到公司同事都在瞪我。

錢如流水般消失，
卻也不知道花到哪。

交通罰單一張一張來，
台北的警察拍照和埋伏的功夫一個比一個厲害。
在山路沒車都會有狗衝出來差點車禍。
直直的路偏偏垃圾車就是沒看到我，硬要迴轉
幾乎被壓在車頭下。

公司同事狀況特別多
明明人數那麼少
卻要玩小學女生才玩的小圈圈遊戲
就不能自己把自己的事做好嗎？
少管別人閒事吧。

手機一直壞
修了四次，到現在還是沒修好
華碩可以再爛一點沒關係啊
被氣到很想把P535往馬路上砸下去

跟家人吵架

沒有錢。

沒有人送我禮物。

騎個腳踏車停紅綠燈也摔車
花大錢搭計程車回家還掉東西
損失慘重。

今天騎車一直被很大聲的喇叭聲嚇到
好幾次
最後在六張犁站那邊
差點忍不住

就想把車停在十字路口
拿出大鎖

看是哪一個王八蛋亂按喇叭
就給他尻下去

還好我還有殘存的一點理智和懦弱
不然現在就是在警察局做筆錄了。

訓練心得

Aug 27th, 2008

連續兩週的禮拜三晚上都去中正運動中心玩了一下射箭
不說訓練，而說玩，是因為真的太久沒射
現在整個動作都沒了，肌肉也退化掉了

不過也趁這個機會重新開始
剛好把以前錯誤的動作矯正過來

之前拉定位頭會歪，前後手沒辦法成一直線
不僅瞄準視線受影響，也不好做延伸

現在重新把頭轉正，定位拉進來一些
就比較好延伸，sighter也不會那麼偏
也對的到弦影了

不過左手有點撐不住
肩膀也很容易聳肩或被壓出來
要抓一下肩膀頂出去的那個點
照理講應該是不用太使力才對

現在定位感覺還不錯
但是還不夠準確大膽
延伸的感覺也還需要抓一下
可能要弄條橡皮筋來練習一下

除了每週三去射箭以外
平常還要盡量做一點重訓
不然這樣回復太慢了

  var text = “”;

  for (var i = 0; i < keywords.length; i++) {
    var re = new RegExp(keywords[i][0].replace(/\s+/, “\s+”), “g”);

發人省思的小故事

Aug 15th, 2008

從前從前，有一個村落，座落在單車鄉中。
單車版的鄉民和樂融融，整天就是騎車聊天，過著幸福快樂的日子。

有一天，其中一位鄉民，賴修文
突然想做一件能代表鄉民的衣服
於是就找了另一位鄉民，蛋m設計衣服
興沖沖的發起了團購

但是賴修文是一個個性莽撞的人
雖然他很古道熱腸，但許多老一輩的鄉民
總覺得他做事不太穩當，擔心交給他會出事。

可是總是要讓年輕人去闖蕩一番，
因此老鄉民也沒說什麼，只是在旁靜靜看著。

隨著團購登記如火如荼的展開，
鄉民們發現怎麼團購方法、價格、廠商都一改再改
搞得大家一頭霧水。

時間咻一下的就過去了，很多人在問：怎麼衣服還沒有消息呢？
在許多人的盼望之下，終於鄉民衣要開始正式製作了！

而這時許多老鄉民們還是靜靜的在旁看著，看著……。

在大事快要底定的前一晚，發生了一件大事，打破了小鎮的寧靜。
原本的設計者：蛋m，只想要一件免費的衣服作為辛苦設計的報酬
沒想到賴修文卻主動問：「你要收多少設計費？有錢大家一起賺。」

蛋m心想：「原來扣掉成本後還有剩這麼多錢，不拿白不拿，我就收點錢好了，可是要收多少呢….？」
正當蛋m猶豫不決的時候，賴修文說了：「要多少，盡管說，我會向鄉民說明清楚。」
但其實賴修文的心理早就有個底了，他心想，加加減減，我想賺的數目扣掉後，大概還有4000元。
4000元再加上一件免費車衣應該夠了。沒關係，我講話講大方一點又不吃虧。
蛋m一時之間，也不曉得該怎麼開價，就隨便開了一件衣服抽20塊的方案。

賴修文這一聽大驚啊！趕忙掐指一算，如果一件衣服你拿20元，那我還賺什麼？
但是大話說在前面，又不好意思反口。
他趕緊飛鴿傳書給所有訂購的鄉民，信上寫著：
「設計的蛋m突然說要多加錢，不然他不給我們設計圖稿。這種事到臨頭才說要加錢的手法令人心寒。」
試圖把錯都推給蛋m。

事情暴發出來後，蛋m當然不堪受辱
趕忙把事情真相告訴大家。

大家一聽，這一個亂啊。人多嘴雜的，更有許多老鄉民在旁冷笑。

過了不久，賴修文貼出沒什麼內容的公告，被大家在上面塗鴉、丟雞蛋。
沒想到賴修文居然用橡皮擦把大家的意見擦掉了！
理由是「沒有購買的人沒有資格發表意見，不要來搗亂！」

可是他卻忘了，雖然不是所有人都有購買，但這件衣服是代表所有單車鄉民的。
只要是鄉民，都有資格發表意見。

在一片撻伐聲中，究竟單車鄉能不能回復以往的和平？
飽受波折的二代鄉民衣又能不能順利出生呢？

….(待續)

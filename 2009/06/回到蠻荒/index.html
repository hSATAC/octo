回到蠻荒

想像你回到蠻荒，那裡沒有火、沒有電、沒有輪子、沒有鐵器。
你知道有這些工具可以用，但是你卻沒辦法用的無力感。
我現在深深體會到了。

我是不知道一個跨兩岸、包含4個程式設計師，一個版面設計師，
至少五個人以上同時在開發的團隊，有什麼資格不用 version control。
連最基本的 SVN 都不肯用。

我的 code 已經被蓋兩次了，讓我有很深的無力感。
從一個習慣用 SVN, BugTracker的地方轉到一個完全不用的團隊，
這不是誰的錯，只是…為什麼有飛機了，我們還要在地上爬呢…？

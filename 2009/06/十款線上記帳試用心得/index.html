十款線上記帳試用心得

Jun 8th, 2009

自從 Palm TX 停用以後就沒記帳了，
最近想來記一下帳，就想找找有沒有好用的線上記帳服務。
最後測試了十款國內外知名的服務。

[線上記帳試用心得]
http://xpenser.com/
介面風格簡單，有一 keyword feature 似乎很好用
支援各種方式記帳包括email twitter msn voice

完全不支援中文，死。

[線上記帳試用心得]
http://www.clearcheckbook.com
介面風格簡單，Dashboard 有widget可自由編排
有許多小工具例如嵌入igoogle, blog等等
有iphone程式可用，並有繁體中文介面！

網站速度慢到爆，貨幣沒有 TWD 可選但是卻有繁體中文= =死

[線上記帳試用心得]
http://money.dj-studio.com.tw
國人開發，似乎很多人推薦

個人開發，信任感不足，首頁就寫網站ip常換，會有連不上主機的情況
看到這邊連 demo 都懶得看了…

[線上記帳試用心得]
http://www.urmoney.tw
國人開發，有點樣子

頁面太花俏，loading太重，網站慢。

[線上記帳試用心得]
http://www.mymoney.somee.com
請您使用 Microsoft Internet Explorer IE 6.0 以上的瀏覽器來穫得最佳效果
掰。

[線上記帳試用心得]
http://gooot.com
大陸人開發，從2004年到現在，算歷史悠久
CSS是pma的，很熟悉
網站速度慢了點，看日誌站主是很有心，但是介面和速度有點難接受
也許等沒選擇了再回頭看看

[線上記帳試用心得]
http://www.homelt.com/
國產，很多人推，完整功能要收VIP費用
看到首頁一堆 table 我就嚇得按下右上角的x了。

[線上記帳試用心得]
http://www.bookkeeper.com.tw
國產，可怕的 layout 讓人望而卻步。

[線上記帳試用心得]
http://www.mint.com
介面和功能似乎都很棒，但是好像US only, 輸入台灣銀行資料都沒用。

[線上記帳試用心得]
http://www.irich.tw
目前為止感覺最ok的，國人開發介面和功能似乎都不錯，
看他們官方部落格應該也很用心。
但是字體很奇怪，有一種很亂的感覺。
網站速度慢，非常大的缺點。

總結：
大陸一些感覺做的不錯的記帳服務例如錢包網之類，
我都沒測了(沒繁體或英文介面看不習慣)。
總之，看來在台灣線上記帳服務還是一個可以切入的點。
商機啊各位～～。

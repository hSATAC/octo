安裝與設定APC

Jul 10th, 2008

安裝與設定APC
———–
linux:
可至PECL下載APC

下載APC後

# tar xvzf APC-2.0.3.tgz 解壓縮
# cd APC-2.0.3
# /usr/bin/phpize
# ./configure –enable-apc
# make
# make install

windows:
可在PECL4Win下載編譯好的dll
請選擇適當版本自行丟到php include目錄中
———-

接著更改php.ini設定

貼上
linux:
extension = apc.so

win:
extension = apc.dll

apc.shm_segments=1
apc.optimization=0
apc.shm_size=128
apc.ttl=7200
apc.user_ttl=7200
apc.num_files_hint=1024
apc.mmap_file_mask=/tmp/apc.XXXXXX
apc.enable_cli=1
apc.rfc1867 = on

重開apache2ctl graceful後，就安裝好了。
可至phpinfo.php檢查是否有正確啟動
或用壓縮檔中的apc.php查看apc資訊
